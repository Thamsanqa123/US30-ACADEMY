# US30 ACADEMY – Financial Trading Mentorship Platform
A financial trading mentorship website, where clients can enroll for mentorship by sending an email with the mentorship program they interested in. The goal was to create a sleek, corporate-finance website that inspires trust, showcases mentorship offerings, and drives enrollment. I built this site because i saw a Forex mentors story(@yandaswings), she was complaining about her DMs being flooded with love messages, instead of people enquiring about the mentorship which was the purpose of her instagram. so i decided to challenge myself and build her a website.

**Live Demo**: https://thamsanqa123.github.io/US30-ACADEMY/

##Tech Stack
- Fronted: vanilla HTML5|CSS3|JAVASCRIPT|FontAwsome
- Backend: Public API (EmailJS)
- Dev Tools: Git/Github
- Deployment: GitHub Pages

##Features

- **Hero Section**: Bold headline with dual CTA buttons ("View Programs" / "Book a Call") and abstract financial background 
- **Pricing Cards**: Two-tier mentorship plans with feature lists, hover animations, and checkout buttons, which lead to the Contact Form 
- **Founder Biography**: Where you get to know more about the founder of the company 
- **Contact page**: Email capture module, a way of communicating with yanda to purchase the course/mentorship 
- **Payment Mockups**: Placeholder buttons for PayPal, card payments, and payment plan options 
- **Mobile-Responsive Navigation**: Hamburger menu with smooth toggle; fully functional across all device sizes(90%) 
- **Sticky Header**: Persistent navigation with "Enroll Now" button for continuous conversion opportunity 
- **Modern Aesthetic**: Dark theme with gold and electric blue accents; clean Inter typography
